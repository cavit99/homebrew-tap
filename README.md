# cavit99 Homebrew Tap

Homebrew formulae for Cavit Erginsoy's tools.

## Install

```sh
brew tap cavit99/tap
brew install parcelcli
```

Or directly:

```sh
brew install cavit99/tap/parcelcli
```

## Formulae

- `parcelcli` — track parcels from the terminal with browser-backed carrier adapters and stable JSON output.
